# People Detection

## Описание
Проект выполняет детекцию людей на видео `crowd.mp4` с помощью YOLO.

## Установка
pip install -r requirements.txt

## Входное видео
Перед запуском положите файл `crowd.mp4` в корень проекта (на одном уровне с `main.py`).

## Запуск
python main.py

## Результат
Видео result.mp4 появится в корне проекта.
